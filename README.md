# extractor_pptx_contents

.pptx 안의 모든 텍스트를 추출하여 엑셀 파일로 내보내는 코드

`import` 폴더에 .pptx 파일들을 넣어놓고 실행시키면 `export` 폴더로 추출결과 파일이 생성된다.
